Coding Challenges

In the treacherous waters of coding, every task is a battle. These challenges are designed to sharpen your skills, test your logic, and prepare you for grand adventures on the Grand Line. Whether it’s a small skirmish or a major clash, every challenge counts!

Challenge Terminology & Samples

Challenge: A puzzle or task requiring you to harness your coding arsenal. Each challenge is a full-blown battle against the horrors of the digital depths.

HTML Challenge: Create a paragraph element that boldly proclaims your coding spirit.

<p>Ahoy, coder!</p>

CSS Challenge: Write a CSS rule that centers text horizontally—like aligning the sails on a pirate ship.

.center-text {
  text-align: center;
}

JavaScript Challenge: Write a function that logs "Hello, World!" to the console—summoning the voice of the digital sea.

function greet() {
  console.log("Hello, World!");
}
greet();

Mini-Challenges

HTML Mini-Challenge: Create an image element for a pirate flag and include alternative text.

<img src="images/pirate-flag.png" alt="Pirate Flag">

CSS Mini-Challenge: Write a rule to change a button's color when hovered—let it shine like gold.

button:hover {
  background-color: #ffcc00;
}

JavaScript Mini-Challenge: Write a function that adds two numbers and logs the sum, proving that even simple math can be heroic.

function add(a, b) {
  console.log(a + b);
}
add(2, 3);

Interactive Challenge: Modify the Function!

Below is a function that adds 2 to a given number. Click the "Transform Function" button and see it change to double the value!

function modifyNumber(num) {
  return num + 2;
}
console.log(modifyNumber(5));
